Welcome to Yacha

Yacha (Yet Another Chat App) is a chat app created as a homework assignment for the course Software Architectures at Budapest University of Technology and Economics. We aimed to create a simple online multiuser chat application demonstrating modern Web technologies such as Babel, Webpack, React and Node.js
